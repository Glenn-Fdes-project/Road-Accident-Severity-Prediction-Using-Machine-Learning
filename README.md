# Road Accident Severity Prediction Using Machine Learning

## 📌 Project Overview
This project predicts the severity of road accidents 
based on weather and road infrastructure conditions 
using Machine Learning models.

The model classifies accident severity into 4 levels:
- Severity 1 → Minor Impact 🟢
- Severity 2 → Moderate Impact 🟡
- Severity 3 → Serious Impact 🟠
- Severity 4 → Fatal Impact 🔴
